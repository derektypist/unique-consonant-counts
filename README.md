# Unique Consonant Counts

Counts the number of unique consonants in a string.  Disregards duplicate consonants of differing cases.  For example, Dad should have 1 unique consonant.

## UX

**Getting Started**

Enter text (e.g. `Dad`) in the text field.  You will see the number of unique consonants as well as the text entered.

**User Stories**

- As a user, I should be able to enter text in the text field
- As a user, I should be able to clear the results by clicking on the Clear button
- As a user, if I keyed in `Dad` in the text field, I should get 1 unique consonant

## Features

Allows the user to enter text in the text field to display the number of unique consonants.

## Technologies

Uses HTML5, CSS3, JavaScript, Bootstrap 5.3.2 and jQuery 3.7.1.
